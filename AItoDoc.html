欢迎使用 AItoDoc 👋
将 AI 输出的 Markdown 一键转为排版精美的 Word / Google Docs 文档
粘贴 Markdown，实时预览
将来自 ChatGPT、Claude、DeepSeek、豆包、Gemini 等 AI 的 Markdown 回复粘贴到左侧输入框，右侧「Word 视图」即时呈现排版效果，包括多级标题、列表、代码块、引用、表格、粗体、斜体等。拖拽中间分隔线可自由调整两栏宽度。
一键复制，粘贴即得成品
点击底部「复制到 Word」按钮，再直接粘贴到 Microsoft Word 或 Google Docs，所有样式（标题层级、加粗、高亮、表格等）完整保留，无需二次调整。底部切换按钮可在 Word 和 Docs 两种目标格式间切换。
LaTeX 数学公式
由 MathJax 驱动，支持行内公式 $…$ / \(…\) 和独占一行的块级公式 $$…$$ / \[…\]，渲染结果可直接复制到 Word。
荧光高亮标注
在右侧预览中选中文字，点击工具栏中的彩色色块（黄 / 绿 / 青 / 粉 / 橙）即可添加荧光高亮；点「清除」去除选中文字的高亮。复制到 Word 后高亮颜色同样完整保留。
书签与定位跳转
在预览中选中任意文字，点工具栏「书签」按钮可附带备注保存为书签；书签列表显示在右侧书签栏，点击名称即可跳转到对应位置。书签随存档一并保存，复制到 Word 时不会包含书签标记。
本地存档，分文件夹管理
点击「存入档案」保存当前文档（含内容、高亮、书签），左侧存档栏支持新建文件夹分类整理、重命名和删除。所有数据保存在本地浏览器，刷新页面不丢失，无需登录。
直接在 Word 视图中编辑
右侧预览区可直接点击输入，像编辑 Word 文档一样修改内容；编辑结果会自动同步回左侧 Markdown，修改与粘贴导出无缝衔接，无需在两侧来回切换。
字体与视图切换
工具栏字体下拉支持切换 Calibri、Times New Roman、Arial、宋体、黑体，实时影响 Word 视图和复制结果。顶部标签可在「分栏」「Markdown」「Word」三种视图间切换，适配不同屏幕尺寸。
